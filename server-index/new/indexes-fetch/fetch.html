0x0013 interacting with urls
Simple section to help you understand how some of the fetch commands work note this section is experimental and is really just here to see if people like the general idea of tools like this being in the script
> fetch documents Fetch documents is a command that will ask you to input a URL and it will crawl that URL for potential documents, if there is one you will get a simple output like this (this output can be glitchy)
RR6> Found document | filename.jpg | RR6> Found document | javascript:void(0); | RR6> Found document | javascript:void(0); | RR6> Found document | javascript:void(0); |
> fetch links this command will attempt to fetch all links on a page via a inputted / given URL example output
+] Found link > | https://developer.aliyun.com/mirror/ > [+] Found link > | https://developer.aliyun.com/mirror/ > [+] Found link 原淘宝 npm 域名即将于5月底停止解析，请切换新域名！ > | < https://developer.aliyun.com/mirror/NPM > [+] Found link 原淘宝 npm 域名即将于5月底停止解析，请切换新域名！ > | < https://developer.aliyun.com/mirror/NPM >
> fetch titles simply fetch the title of a website using -> google.com example output
